QCapp v11 deployment checklist

1. Upload every file in this folder to the root of the proto-qc repository.
2. Replace index.html, sw.js, manifest.webmanifest, README.md, and existing icon files.
3. Commit the changes.
4. Wait for GitHub Pages deployment to finish.
5. Test https://apdelany1.github.io/proto-qc/?v=11
6. On iPhone, completely close and reopen the home-screen app.
7. Verify one page contains Row Details, Photos, Sample Size, and Inspection Counts.
8. Rename a temperature and a defect, switch tabs, return, and confirm the names and counts remain.
